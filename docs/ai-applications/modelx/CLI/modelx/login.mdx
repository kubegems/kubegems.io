## Login model repository

--- 

登录模型仓库，如果 modelxd 服务未开启认证，可跳过 token 阶段，直接回车即可

```
$ modelx repo list

+--------+----------------------------+
| NAME   | URL                        |
+--------+----------------------------+
| modelx | https://modelx.kubegems.io |
+--------+----------------------------+

$ modelx login modelx 
Token:
Login successful for modelx
```
